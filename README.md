# Pet Resume Generator for Rental Applications

A simple MVP built with Next.js + Tailwind CSS.

## Features
- English-only form
- Rule-based pet resume generation
- Rule-based landlord intro note generation
- Missing-item reminders
- Copy buttons for both outputs
- PDF export via browser print dialog
- Optional local photo preview

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build
```bash
npm run build
npm run start
```
